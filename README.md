wp-scholar
==========

Creates post types that match scholarly content types.

Content Types
=============
- Feeds
- Publications
- People (undergrads, interns)
- Gallery
- Courses
- Graduate Studies

Installation
============
- Download the plugin through the WordPress plugin installer
- Activate
- Go to Dashboard > Content Types to activate the relevant content types for your site
- Go to Plugins > WP Scholar to configure the plugin
- Plugin comes with Zurb Foundation-compatible fallback themes, but you may wish to design your own. Copy any post type template into your own template directory and modify per your needs.
